drake
=====

**Currently a (W)ork (I)n (P)rogress**

Requirements
------------

* Ruby 1.9.*
* Redis (default 127.0.0.1:6379)

Run
---

    git clone https://github.com/RealSelf/drake.git
    cd drake
    bundle install
    ruby app.rb
  
Test
----

    cd drake
    rspec --format documentation
